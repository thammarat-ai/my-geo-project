# my-geo-project

**my-geo-project** เป็นโปรเจกต์ตัวอย่างสำหรับทดลองวิเคราะห์และแสดงผลข้อมูลภูมิสารสนเทศ (Geospatial Data) เช่น พิกัดสถานที่ เส้นทางการเดินทาง หรือขอบเขตพื้นที่ต่างๆ โดยใช้เครื่องมือโอเพนซอร์สยอดนิยมอย่าง GeoPandas, PostGIS และ Leaflet จุดประสงค์หลักคือให้สามารถนำข้อมูลดิบ เช่น ไฟล์ CSV, Shapefile หรือ GeoJSON มาประมวลผลและนำเสนอผ่านหน้าเว็บแบบโต้ตอบได้อย่างสะดวกและรวดเร็ว

ในโปรเจกต์นี้จะมีโครงสร้างโฟลเดอร์แยกชัดเจน เช่น `data/` สำหรับเก็บข้อมูลดิบ, `notebooks/` สำหรับ Jupyter Notebook ที่ใช้ทดลองวิเคราะห์, `api/` สำหรับโค้ดเบื้องหลัง (backend) ในการให้บริการข้อมูลเชิงพื้นที่ และ `frontend/` สำหรับเว็บแผนที่อินเตอร์แอคทีฟ ผู้ใช้สามารถทดลองสร้างเลเยอร์แผนที่หลายชั้น ซ้อนข้อมูลเช่น จุดร้านค้า เส้นทางรถเมล์ หรือโซนความเสี่ยง แล้วเปิด/ปิดแต่ละเลเยอร์เพื่อเปรียบเทียบข้อมูลได้

ตัวอย่างการใช้งานพื้นฐานคือ การนำข้อมูลพิกัดร้านค้าทั้งหมดในเมืองหนึ่งมาวิเคราะห์หาพื้นที่ที่มีความหนาแน่นสูง จากนั้นสร้าง Heatmap แสดงบนเว็บแผนที่ พร้อมฟังก์ชันค้นหาและกรองข้อมูล (filter) ตามประเภทกิจการหรือช่วงเวลาเปิด–ปิดร้าน โปรเจกต์นี้จึงเหมาะกับทั้งสาย Data, นักพัฒนาเว็บ, และผู้ที่ต้องการต้นแบบ (prototype) ระบบสารสนเทศภูมิศาสตร์ (GIS) เบื้องต้นสำหรับต่อยอดเป็นระบบขนาดใหญ่ในภายหลัง
